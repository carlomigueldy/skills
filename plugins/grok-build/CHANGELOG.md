# Changelog

All notable changes to `grok-build` are documented here. This plugin is
released independently through release-please and uses Conventional Commits.

## 0.1.0 - 2026-07-22

### Features

* **grok-build:** initial Grok Build orchestration kit — `/ultracode` skill,
  `ship-feature` Rhai workflow, and tiered personas/roles
  (`architect`, `sweeper`, `implementer`, `reviewer`) with install script
  targeting `~/.grok/`.
